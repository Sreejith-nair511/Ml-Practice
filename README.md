# Machine Learning Lab Practice

A collection of Machine Learning laboratory programs, datasets, notebooks, and practice exercises implemented as part of the B.E. Information Science and Engineering curriculum.

## Overview

This repository contains practical implementations of fundamental Machine Learning concepts including:

* Data Preprocessing
* Data Visualization
* Classification Algorithms
* Clustering Techniques
* Regression Models
* Dimensionality Reduction
* Model Evaluation
* Neural Networks
* Unsupervised Learning

The repository is intended for academic learning, lab preparation, internal assessments, and examination practice.

## Technologies Used

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Lab Programs

1. Data Preprocessing and Cleaning
2. Exploratory Data Analysis
3. K-Nearest Neighbors (KNN)
4. Decision Tree Classification
5. Naive Bayes Classification
6. Support Vector Machine (SVM)
7. Linear Regression
8. Principal Component Analysis (PCA)
9. K-Means Clustering
10. Gaussian Mixture Model (GMM)

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/ml-lab-practice.git
```

### Install Dependencies

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```
